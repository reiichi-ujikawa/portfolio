# ポートフォリオ概要
採用人事として長年、応募者対応や面接調整、採用管理などの業務に従事してきました。

業務を進める中で、定型業務や繰り返し作業の多さに課題を感じ、Power Automateを活用した業務自動化に取り組む機会がありました。

実際に自動化によって業務効率が改善される経験を通じて、業務改善そのものに大きなやりがいを感じ、RPA領域へ本格的に挑戦したいと考えるようになりました。

現在はPower Automateの実践的な学習に加え、UiPathやPythonの学習、ポートフォリオ作成にも取り組み、業務理解を強みとしたRPAエンジニアを目指しております。
